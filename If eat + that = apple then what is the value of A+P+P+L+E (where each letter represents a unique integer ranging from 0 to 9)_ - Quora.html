If eat + that = apple then what is the value of A+P+P+L+E (where each letter represents a unique integer ranging from 0 to 9)?

The ans is 12

Let say.
                   e a t
            +   t h a t
             ---------------------
               a p p l e

For this conditions to satisfy t=9 because to get a 5 digit number by adding 3&4 digit number first value of 4 digit number should be 9 then automatically a=1.

Now 1000th & 100th place  should be same as according to value we took earlier 1000th place value in 0 then 100th place should be 0 also and this can be achieve through

6+4
8+2
5+5

According to above situation
E= t+t
Means 9+9=18
That is e=8
So h=2

                     8 1 9
               + 9 2 1 9
             -------------------
               1 0 0 3 8

A = 1
P = 0
L = 3
E = 8

A + P + P + L + E
1 + 0 + 0  + 3 + 8 = 12
